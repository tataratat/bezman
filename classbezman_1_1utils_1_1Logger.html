Detailed Description

Provides functionality for output information.

The logger class provides functionality to write log files and to format user information, warnings and errors for the console output. There only ever exists a single instance of the Logger class, as it is provided as a singleton object. The Level of information that is provided by the logger can be changed at runtime, however, the logger needs to be activated with a Preprocessor flag at compile time
